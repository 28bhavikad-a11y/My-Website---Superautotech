Upload each with the **exact** filename (all lowercase, `.png`). Until a file is
here with the right name, that photo shows as broken.

## How to upload (GitHub website — about 2 minutes)
1. Open your repo on github.com and click into the `images` folder.
2. Click **Add file -> Upload files**.
3. Drag your photos in. Rename each to match the table above exactly before/after dragging.
4. Click **Commit changes**.

After committing, refresh the website and the photos appear.

Tip: keep each photo under ~500 KB so pages load fast.
