# Images for the website

The website pages look for the photo files listed below **inside this `images/` folder**.
Until a file is here with the **exact** name, that photo shows as a broken image.

Upload each file with the exact filename (all lowercase, `.png`):

| Filename                   | What it should be                                  | Shown on |
|----------------------------|----------------------------------------------------|----------|
| `logo-sat.png`             | Your Super Autotech logo (header + footer)         | Every page |
| `logo.png`                 | Logo for Google/social SEO data (can be same logo) | index (SEO only, not visible) |
| `mlv-sheet.png`            | MLV roll — main hero product photo                 | Home, Automotive, Mass Loaded Vinyl |
| `mlv-cutaway.png`          | MLV sheet / cross-section detail shot              | Home, product pages |
| `install-wall.png`         | Installed shot — residential room                  | Home |
| `family-room.png`          | Installed shot — wall framing                       | Home |
| `soundproofing-collage.png`| Applications overview collage                       | Home |
| `mlv-rolls-pallet.png`     | Pallet of MLV rolls — manufacturing (also the social-share preview image) | Home / social previews |
| `your-photo.png`           | Placeholder photo used by the new-product template  | _product-template.html |

## How to upload (GitHub website, easiest)
1. Open your repo on github.com and go into the `images` folder.
2. Click **Add file → Upload files**.
3. Drag in your photos, making sure each filename matches the table above exactly.
4. Click **Commit changes**.

After they're committed, refresh the website and the photos will appear.

Tip: keep each photo under ~500 KB (resize/compress large camera photos) so pages load fast.
